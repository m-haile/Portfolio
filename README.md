# Portfolio

Ce dépôt contient le portfolio personnel de Mahlet Haile Gebru, développeuse web junior basée à Limoges.

## À propos

Ce site vitrine présente son profil, ses compétences, ses projets, son parcours et ses coordonnées. Il a été conçu pour offrir une présentation moderne, claire et accessible de son parcours professionnel.

## Fonctionnalités

- page d'accueil avec une section hero,
- navigation interne rapide,
- sections « À propos », « Compétences », « Projets », « Parcours » et « Contact »,
- design responsive et attention portée à l'accessibilité,
- lien de téléchargement du CV.

## Structure du projet

- [index.html](index.html) : structure principale du portfolio
- [css/style.css](css/style.css) : styles, thème sombre et mise en page responsive
- [js/script.js](js/script.js) : mise à jour du lien actif dans la navigation au défilement
- [assets/](assets/) : fichiers multimédias et CV

## Démarrage

1. Clonez ce dépôt.
2. Ouvrez [index.html](index.html) dans votre navigateur.
3. Pour une prévisualisation plus pratique, vous pouvez utiliser une extension comme Live Server.

## Technologies utilisées

- HTML5
- CSS3
- JavaScript
- Google Fonts
